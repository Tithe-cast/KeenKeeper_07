KeenKeeper

Your personal shelf of meaningful connections.

KeenKeeper helps you stay in touch with the people that matter most by tracking interactions, setting contact goals, and reminding you when friendships need attention.

Technologies Used

- React.js — Component-based UI
- React Router DOM — Client-side routing
- Tailwind CSS — Utility-first styling
- Recharts — Friendship analytics pie chart
- React Hot Toast — Toast notifications
- Lucide React — Icon library
- Vite — Build tool

Key Features

1. Friend Cards Grid — Visual overview of all friends with status badges (On Track, Almost Due, Overdue)
2. Quick Check-In— Log Call, Text, or Video interactions directly from a friend's detail page
3. Timeline & Analytics — Full interaction history with filtering + pie chart stats by interaction type
